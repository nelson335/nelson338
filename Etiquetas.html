Etiquetas HTML, CSS y JavaScript Esenciales

¿Qué son las etiquetas HTML?

Las etiquetas HTML son los elementos fundamentales que dan estructura y significado al contenido de una página web. Actúan como instrucciones que le indican al navegador cómo mostrar cada parte del contenido, desde títulos y párrafos hasta imágenes y enlaces.

Cada etiqueta tiene un propósito específico y sigue una sintaxis estandarizada, permitiendo crear documentos web bien organizados y accesibles. Juntas, forman el esqueleto de cualquier sitio web moderno.

Etiquetas Básicas HTML

Estas son las etiquetas fundamentales que todo documento HTML debe contener. Proporcionan la estructura básica y la información esencial que los navegadores necesitan para interpretar correctamente tu página web.

<!DOCTYPE>
Declara el tipo de documento y la versión de HTML que se está utilizando. Es la primera línea que debe aparecer en cualquier documento HTML.
<!DOCTYPE html>
<html>
El contenedor principal que engloba todo el documento HTML. Todas las demás etiquetas deben estar dentro de esta.
<html lang="es">...</html>
<head>
Contiene metadatos sobre el documento, como el título, enlaces a hojas de estilo, scripts y otra información que no se muestra directamente en la página.
<head>...</head>
<meta>
Proporciona metadatos sobre el documento HTML, como descripción, autor, palabras clave y configuración de la vista.
<meta charset="UTF-8">
<link>
Establece la relación entre el documento actual y un recurso externo, como hojas de estilo CSS.
<link rel="stylesheet" href="styles.css">

Etiquetas de Contenido HTML

Estas etiquetas definen la estructura del contenido visible de tu página web. Organizan el texto, imágenes y otros elementos que los usuarios ven y con los que interactúan.

<body>
Contiene todo el contenido visible de la página web, como texto, imágenes, videos y otros elementos multimedia.
<body>...</body>
<h1>-<h6>
Definen encabezados de diferentes niveles de importancia (h1 es el más importante, h6 el menos). Esenciales para la estructura semántica y SEO.
<h1>Título Principal</h1>
<p>
Define un párrafo de texto. Automáticamente añade espacio antes y después del texto para separarlo de otros elementos.
<p>Este es un párrafo de ejemplo.</p>
<span>
Contenedor en línea genérico para contenido que no tiene significado semántico, útil para aplicar estilos.
<span class="resaltar">texto</span>
<ul>, <ol>, <li>
Crea listas no ordenadas (ul), ordenadas (ol) y elementos de lista (li).
<ul><li>Item 1</li></ul>

Etiquetas de Enlace y Multimedia

Estas etiquetas permiten conectar tu página con otros recursos, ya sean páginas externas, secciones internas o elementos multimedia que enriquecen la experiencia del usuario.

<a>
Crea hipervínculos a otras páginas web, archivos, ubicaciones dentro de la misma página, o cualquier otra URL.
<a href="https://ejemplo.com">Visitar Ejemplo</a>
<img>
Inserta una imagen en el documento. Requiere el atributo src para especificar la ruta de la imagen y alt para texto alternativo.
<img src="imagen.jpg" alt="Descripción">
<video>
Inserta contenido de video en el documento. Soporta múltiples formatos como MP4, WebM y Ogg.
<video src="video.mp4" controls></video>
<audio>
Inserta contenido de audio en el documento. Soporta formatos como MP3, WAV y Ogg.
<audio src="audio.mp3" controls></audio>

Propiedades CSS Esenciales

CSS (Cascading Style Sheets) es el lenguaje que controla el diseño visual de las páginas web. Estas son algunas de las propiedades más importantes:

color
Define el color del texto. Puede ser un nombre de color, código hexadecimal, RGB o HSL.
color: #333;
color: red;
color: rgb(255, 0, 0);
font-size
Controla el tamaño del texto. Se puede especificar en px, em, rem, % u otras unidades.
font-size: 16px;
font-size: 1.2em;
margin
Establece el espacio exterior alrededor de un elemento. Controla la separación entre elementos.
margin: 10px;
margin: 10px 20px;
padding
Establece el espacio interior de un elemento, entre su contenido y su borde.
padding: 15px;
padding: 10px 20px;
display
Controla cómo se muestra un elemento (en bloque, en línea, flex, grid, etc.).
display: block;
display: flex;
display: none;

Conceptos Básicos de JavaScript

JavaScript es el lenguaje de programación que permite agregar interactividad a las páginas web. Estos son algunos conceptos fundamentales:

Variables
Contenedores para almacenar datos. Se declaran con let, const o var.
let nombre = "Juan";
const PI = 3.1416;
Funciones
Bloques de código reutilizables que realizan una tarea específica.
function saludar() {
  console.log("Hola");
}
Eventos
Acciones que ocurren en el navegador (clic, teclado, carga, etc.) a las que puedes responder.
document.getElementById("btn").addEventListener("click", function() {
  alert("Hiciste clic!");
});
DOM
Document Object Model - Representación del documento HTML que permite modificarlo con JavaScript.
document.querySelector(".clase")
elemento.textContent = "Nuevo texto"
Condicionales
Estructuras que permiten ejecutar código bajo ciertas condiciones.
if (edad >= 18) {
  console.log("Mayor de edad");
}
