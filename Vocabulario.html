Vocabulario Técnico

1. HTML

HTML (HyperText Markup Language) es el lenguaje de marcado básico utilizado para crear y estructurar el contenido de una página web. Define la estructura y el significado del contenido, como texto, imágenes y enlaces. Los navegadores web interpretan el código HTML para mostrar la página en el dispositivo del usuario.

2. CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para controlar la apariencia de una página web, separando el contenido (HTML) de la presentación visual. Permite definir estilos, colores, diseños y responsividad para los elementos HTML.

3. JavaScript

JavaScript es un lenguaje de programación de scripting interpretado que se utiliza principalmente para la creación de contenido dinámico y la interacción en páginas web. Permite añadir funciones interactivas y mejorar la experiencia del usuario.

4. Etiquetas HTML

Las etiquetas HTML son códigos que estructuran el contenido de una página web y le indican al navegador cómo mostrarlo. Son el lenguaje de marcado que define la estructura y el contenido de una página web.

5. Extensiones VS Code

Las extensiones VS Code son complementos que añaden funcionalidades y personalizan la experiencia de desarrollo en Visual Studio Code, permitiendo expandir sus capacidades según las necesidades del programador.

6. Sangría

La sangría en programación se refiere al espacio en blanco que se añade al inicio de una línea de código para indicar la estructura y jerarquía del código. Ayuda a hacer el código más legible y facilita la comprensión de su estructura.

7. Página Web

Una página web es un documento electrónico, generalmente escrito en HTML, que se puede acceder a través de Internet mediante un navegador web. Contiene información y puede incluir texto, imágenes, enlaces y otros elementos multimedia.

8. Framework

Un framework es una estructura o conjunto de herramientas y componentes reutilizables que simplifican el desarrollo de aplicaciones o proyectos de software, proporcionando una base sobre la cual construir.

9. Aplicación

Una aplicación es un programa de software diseñado para realizar una tarea específica, ya sea en dispositivos móviles, computadoras de escritorio o web. Puede ser desde una simple calculadora hasta complejos sistemas empresariales.

10. Usuario

Un usuario es una persona o entidad que utiliza un producto, servicio, sistema o plataforma. En términos sencillos, es quien hace uso de algo, ya sea de manera habitual u ocasional.

11. Interfaz

Una interfaz es el punto de interacción y comunicación humano-computadora en un dispositivo. Puede ser gráfica (GUI), de línea de comandos (CLI) o táctil, entre otras.

12. Navegadores

Los navegadores son programas que permiten acceder a la información en la World Wide Web, interpretando y mostrando el contenido de las páginas web. Ejemplos incluyen Chrome, Firefox y Safari.

13. Base de datos

Una base de datos es un sistema organizado para almacenar, recuperar y gestionar información de forma estructurada. Permite el manejo eficiente de grandes volúmenes de datos con integridad y seguridad.

14. Atributo

Un atributo es una característica o propiedad que describe un objeto, elemento o archivo. Puede ser utilizado para identificar, describir o modificar un elemento, y puede tener un valor específico asociado.

15. Entidad

Una entidad es una abstracción que representa un objeto o concepto del mundo real del que se almacena información. En bases de datos, es una tabla que contiene datos sobre un tipo específico de elemento.

16. Protocolo

Un protocolo es un conjunto de reglas, convenciones o estándares que definen cómo los dispositivos, sistemas o aplicaciones se comunican entre sí en una red. Ejemplos incluyen HTTP, TCP/IP y FTP.

17. Software

El software son los componentes no físicos de una computadora o dispositivo electrónico, como programas, instrucciones y datos. Es la parte lógica que permite el funcionamiento del hardware.

18. Blog

Un blog es una página web o sección dentro de una página web que publica regularmente artículos, noticias y opiniones sobre temas relacionados con la tecnología, software, hardware, programación y otros aspectos tecnológicos.

19. Industria del software

La industria del software es el sector económico que abarca la creación, distribución y mantenimiento de programas informáticos y aplicaciones. Impulsa la innovación y la transformación digital en diversas áreas.

20. Tecnologías modernas

Las tecnologías modernas son aquellas que han surgido y evolucionado en la era contemporánea, marcadas por avances en la ciencia, la ingeniería y la informática, como IA, blockchain y computación cuántica.

21. Buster

Buster en informática puede referirse a varias cosas, aunque la más común es una herramienta para el reconocimiento de correo electrónico.

22. Tailwind

Tailwind es un framework de CSS que se centra en la utilidad sobre el estilo, proporcionando clases de bajo nivel que se pueden componer para construir diseños directamente en el marcado.

23. CEO

CEO (Chief Executive Officer) es la persona que se encarga de dirigir la empresa. Es el máximo responsable a nivel operativo y toma las decisiones estratégicas clave.

24. API

API (Application Programming Interface o Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permite que diferentes aplicaciones de software se comuniquen entre sí.

25. Online

Online significa que algo está conectado a una red de datos o de comunicación, generalmente Internet, y está disponible para su uso inmediato.

26. Sitio web

Un sitio web es una colección de páginas interconectadas que se encuentran en Internet y pueden ser accesibles a través de una dirección única (URL).

27. Metodologías ágiles

La metodología ágil es un enfoque de gestión de proyectos que se centra en la flexibilidad, la colaboración y la entrega continua de valor, priorizando la adaptabilidad a los cambios y la satisfacción del cliente.

28. Desarrollo web

El desarrollo web se refiere al proceso de creación y mantenimiento de sitios web.

29. Bootstrap

Bootstrap es un framework frontend de código abierto y gratuito que facilita la creación de sitios web y aplicaciones web responsivas.

30. UX/UI

UX/UI design (Diseño UX/UI) es un campo que se centra en la experiencia del usuario (UX) y la interfaz de usuario (UI) de productos digitales.

31. Web Performance

Web Performance se refiere a la velocidad con la que se descargan y se muestran las páginas web en el navegador del usuario.

32. Accesibilidad global

La accesibilidad global se refiere a diseñar y desarrollar sitios web, herramientas y tecnologías para que sean utilizables por todas las personas, incluyendo aquellas con discapacidad.

33. Google

Google es una empresa que ofrece una amplia variedad de servicios, principalmente basados en internet.

34. Interactividad

La interactividad se refiere a la capacidad de un sistema, dispositivo o contenido de responder a la entrada o las acciones de un usuario, creando una comunicación bidireccional.

35. Pseudocódigo

El pseudocódigo es una forma no oficial de representar algoritmos o instrucciones de programación usando un lenguaje similar al español, pero sin las reglas estrictas de un lenguaje de programación.

36. Scrum

Scrum es un marco ágil utilizado por equipos para gestionar el desarrollo de productos de manera iterativa e incremental.

37. Kanban

Kanban es un método de gestión visual que ayuda a los equipos a visualizar su flujo de trabajo, limitar el trabajo en progreso (WIP) y mejorar la eficiencia.

38. Testing

Testing se refiere al proceso de evaluar la calidad y funcionamiento de un software para identificar defectos y asegurar su correcto funcionamiento.

39. Feedback

Feedback es el proceso de proporcionar información a alguien sobre su desempeño o resultados con el objetivo de lograr mejoras futuras.

40. Administrador

Un administrador es una persona que planifica, organiza y supervisa la ejecución de proyectos, asegurándose de que se completen a tiempo, dentro del presupuesto y cumpliendo con los requisitos establecidos.

41. Código Fuente

El código fuente son las instrucciones escritas que un programador utiliza para crear un programa informático.

42. Internet

Internet es una red global de redes de computadoras que permite la comunicación y el intercambio de información entre dispositivos a nivel mundial.

43. Computadora

Una computadora es una máquina programable que ejecuta una serie de comandos para procesar los datos de entrada, obteniendo convenientemente información que posteriormente se envía a las unidades de salida.

44. FTP

FTP es un protocolo de red que permite la transferencia de archivos entre un cliente y un servidor a través de una red TCP/IP, como Internet.

45. Estándares de calidad

Los estándares de calidad son documentos que establecen requisitos, especificaciones, guías o características que se utilizan para garantizar que los materiales, productos, procesos y servicios sean adecuados para su propósito.

46. Simplicidad en la web

La simplicidad en la web se refiere a crear diseños y experiencias de usuario que son fáciles de entender, usar y navegar.

47. Jerarquía visual

La jerarquía visual es la organización de elementos de diseño en un orden que guía la vista del espectador y le permite percibir la importancia de cada elemento.

48. Navegación intuitiva

La navegación intuitiva se refiere a la facilidad con la que un usuario puede interactuar y navegar a través de una interfaz, sin tener que aprender o entender instrucciones explícitas.

49. Consistencia

La consistencia se refiere a mantener la uniformidad y coherencia en diversas áreas de desarrollo, como el código, la arquitectura de software, el diseño de interfaces y la colaboración en equipo.

50. HTTPS

HTTPS significa "HyperText Transfer Protocol Secure", la versión segura de HTTP. Utiliza el cifrado para proteger la información que se transfiere entre un navegador web y un servidor, garantizando que los datos no sean interceptados ni alterados durante el viaje.

51. W3C

W3C es un consorcio internacional que genera recomendaciones y estándares que aseguran el crecimiento de la World Wide Web a largo plazo.

52. Desarrollo frontend

El desarrollo frontend se enfoca en crear y mantener la interfaz de usuario de un sitio web o aplicación, lo que el usuario ve e interactúa con él.

53. RED LAN

Una RED LAN es una red de computadoras y otros dispositivos conectados en un área geográfica pequeña, generalmente dentro de un edificio o campus.

54. IP

Una IP es un número único que identifica un dispositivo dentro de una red que utiliza el protocolo de Internet.
